# SPECIMEN OF THOUGHT

현대미술의 질문과 고대 철학자의 관점을 연결한 모바일 우선 정적 웹사이트입니다.

## 실행

빌드 도구나 외부 의존성이 없습니다. `index.html`을 브라우저에서 열면 동작합니다.

로컬 서버로 확인하려면 다음 명령을 사용할 수 있습니다.

```powershell
python -m http.server 8080
```

그 뒤 `http://localhost:8080`으로 접속합니다.

## 구현 범위

- 오프닝, 안내, 최초 판단
- 철학자 3명 선택과 분기 대화
- CUBIST VIEW 비교 화면
- 최종 판단, 판단 변화, 생각 표본 카드
- `localStorage` 기반 진행 상태 복구
- 모바일/태블릿/데스크톱 반응형
- 키보드 조작과 reduced-motion 대응

대화는 실제 발언이 아닌 교육 목적의 재구성입니다. 외부 작품 이미지나 AI API는 사용하지 않습니다.
